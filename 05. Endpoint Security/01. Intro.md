# Endpoint Security Controls

- Antivirus/Anti-malware: Pattern and signatures based scans
- EDR : real-time monitoring and response which is agent based deployment, It detect process, file, registry and network activity
- XDR:  Integration of Multiple security controls and telemetry which runs on automated responses
- DLP: control and protect data in transit, rest and processing
- UBA(User Behavior analytics): Identify potential Insider Threats
- Host-Based firewall : firewall that control incoming and outgoing traffic on a host

**End Point Security Monitoring**
- Process execution
- File System Changes(File Integrity Monitoring)
