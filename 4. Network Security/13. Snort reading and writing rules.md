- Snort run as service in BG 24/7 for monitoring but network webs but can be run manually for testing

- Community rules: rules submitted by open source community
	- can be used as a refrence point to make our own rules and run rules which from created by post attack victims
- Snort Logs: all the logs are saved in /var/log/snort
	- to run snort in such way that it saves the logs for SIEM 
		- run snort in alert mode full/fast   == $ sudo snort -A fast -l {dir to save logs} -i {NI} -c /etc/snort.conf -q
