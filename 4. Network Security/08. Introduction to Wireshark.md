GUI based utility for network troubleshooting and analysis and most widely used

It can also capture live and static pcap and advantageous over TCPdump

- For real time traffic capture on high speed network=TCPdump
- Detailed packet analysis can be resource-intense = Wireshark

**Wireshark For linux** 
- $ sudo apt install wireshark
	- for advance releases= sudo add-apt-repository ppa:wireshark-dev/stable
	- $sudo apt update
	- sudo apt install wireshark
