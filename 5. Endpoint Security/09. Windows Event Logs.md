- Updates logs
- User logs
- Software Crash logs

These logs are easily forwarded to SIEM for centralized logging

	- Setup: application setup and changes
	- System: all logs related to drivers or system fails

**Some Cli Utilities for Query of  System Events
- wevutil: Query, Export, Archive Query locally and remotely 
	- wevutil qe security /c:5 /f:text /rd:true
		- c:5= shows only 5 entries
		- f:text=format(wevutil comes in binary)
		- rd:true= shows recent one first
	- can also filter with Event Ids
		- wevutil qe security /c:5 /f:text /rd:true /q:"*[System[(EventID=4720)]]"

- Get-WinEvent for powershell
