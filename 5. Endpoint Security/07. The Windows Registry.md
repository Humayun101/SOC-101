Registry = A hierarchical data-base within windows that store keys, sub-keys and option to store windows setting

regedit is the utility to see and modify windows Registry
- Registry is divided into several hives
- Each hive contain a specific set of data
	- <img src="Pasted image 20251109170546.png">
	
	- Each hive starts with HKEY(Handle to register key)

**Different Hives**
- HKEY_CLASSES_ROOT; Stores info about registered application
- HKEY_Current_user; Contain user specific settings and configuration(The configuration here point to HKEY_USER hive)
- HKEY_Local_Machine; Stores things about configuration data for the local computer, including information about the operating system, hardware, device drivers, and installed software used by all users
- HKEY_USERS: Contains all Users and their Information
- HKEY_Current_config: Info about current hardware profile(A pointer toward HKEY_Local machine)

*We can also edit registry entry using cmd.exe by "reg add 'location' /v and condtions"*
